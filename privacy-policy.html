FieldD Extension Privacy Policy

Last Updated: March 19, 2024

Introduction

This privacy policy explains how the FieldD Extension ("we", "our", or "us") handles your data when you use our Chrome extension.

Data Collection and Usage

Our extension:

  • Only accesses job data from the FieldD website (fieldd.com)
  • Uses Google Maps API to calculate drive times between jobs
  • Does not store any personal information
  • Does not share any data with third parties

Required Permissions

Our extension requires the following permissions:

  • Access to fieldd.com - to read job information
  • Google Maps API - to calculate drive times
  • Storage - to temporarily store job selections

Data Security

We take data security seriously:

  • All data processing happens locally in your browser
  • No data is stored on our servers
  • No personal information is collected

Third-Party Services

We use the following third-party services:

  • Google Maps API - for route calculations
  • These services have their own privacy policies

Contact

If you have any questions about this privacy policy, please contact us at:

Email: support@fieldd.com

Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.
